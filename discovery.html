LOCAL DISCOVERY RECORD

This page lets locals identify what the next Bellbrook issue should become.

How this works

Issue 3 starts as discovery. Locals submit accounts about what they think needs attention. If enough archive receipts point to the same issue, that issue can be promoted into the active Issue 3 module. Discovery then moves forward and becomes Issue 4.

Promotion logic

  • Many separate residents identify the same issue.
  • The accounts point to the same place, authority, pattern, or impact.
  • The issue has enough detail to define what action is being asked for.
  • The archive can show why the issue has become a priority.

What this form does not do

It does not decide the issue in advance. It does not make allegations. It collects local accounts so a pattern can be identified before the next dedicated module is built.
